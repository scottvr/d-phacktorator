# prereqs
install the dphackt library
e.g.:
```bash
cd ..
python ./setup.py
cd -
```
or possibly by now
```
pip install phackt
```

# Usage:
```
python phackt.py --data_dir ./data --output_dir out --cache_dir ./cache --window_size 30 --threshold 0.7 --date_column_map columns.json
```
